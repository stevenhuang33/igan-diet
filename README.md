# 🥗 IgA Nephropathy Stage 3 CKD Dietary Guide

A comprehensive, user-friendly dietary planning resource for patients with IgA Nephropathy and Stage 3 Chronic Kidney Disease.

## 🎯 Purpose

This web application provides evidence-based dietary guidance for patients managing IgA Nephropathy and Stage 3 CKD, helping them make informed food choices to protect kidney health.

## ✨ Features

### 📋 Comprehensive Food Lists
- **Foods to Eat**: Kidney-friendly options with portion sizes
- **Foods to Limit/Avoid**: High-potassium, high-phosphorus, and high-sodium foods
- Color-coded system (Green/Yellow/Red) for easy identification

### 📊 Daily Nutritional Targets
- Protein (0.8-1.0 g/kg body weight)
- Sodium (< 2000mg/day)
- Potassium (2000-3000mg)
- Phosphorus (800-1000mg)
- Fluid intake guidelines
- Caloric recommendations

### 🍽️ Sample Meal Plan
- Complete daily meal plan (breakfast, lunch, dinner, snacks)
- Portion sizes and nutritional estimates
- Hydration tips

### 💡 Practical Tips & Guidelines
- Cooking tips (leaching potassium, salt-free seasoning)
- Eating out strategies
- Food label reading
- Important warnings (salt substitutes, supplements)

### 🔍 Search & Filter
- Real-time search across all foods
- Expandable/collapsible categories
- Mobile-responsive design
- Print-friendly format

## 📚 What's Covered

### Foods to Eat (Kidney-Friendly)
- Lean proteins (chicken, fish, egg whites)
- Low-potassium vegetables (cauliflower, green beans, cabbage, bell peppers)
- Low-potassium fruits (apples, blueberries, grapes, strawberries)
- Kidney-friendly grains (white rice, white bread, plain pasta)

### Foods to Limit/Avoid
- **High-Potassium**: Bananas, potatoes, tomatoes, oranges, spinach
- **High-Phosphorus**: Dairy, nuts, whole grains, dark sodas
- **High-Sodium**: Processed meats, canned foods, restaurant food, pickles

### Nutritional Targets (Stage 3 CKD)
- **Protein**: 0.8-1.0 g/kg/day (moderate restriction)
- **Sodium**: < 2000mg/day
- **Potassium**: 2000-3000mg/day
- **Phosphorus**: 800-1000mg/day
- **Fluids**: Individual (based on urine output)
